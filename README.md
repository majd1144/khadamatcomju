Khdamatcom is a Full-Stack graduation project designed to connect freelancers with people seeking services or jobs.  
The platform is developed using React for the frontend, Express.js for the backend with RESTful APIs, and PostgreSQL for the database.

---

Features:
- User registration and login for both freelancers and clients  
- Display available services for each category  
- Add and manage personal services for freelancers  
- Search and filter services by type and location  
- Data management through RESTful API  



Tech Stack:
- Frontend: React.js, HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database:PostgreSQL  
- API: RESTful API  
- Version Control:Git, GitHub
